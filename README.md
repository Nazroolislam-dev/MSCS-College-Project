# 📊 MSCS College Projects
A collection of advanced academic and personal projects focusing on Mathematics, Statistics, and Computer Science.

## 🚀 Featured Project: Logistics Weather Pro Monitor (DSS)
This is a **Decision Support System (DSS)** designed to assist logistics managers in making data-driven choices based on real-time environmental factors.

### 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3 (Glassmorphism design), JavaScript
* **Concepts:** Risk Assessment, Data Visualization, Real-time Monitoring

### 📈 Key Features
* **Risk Monitor:** Evaluates weather conditions to provide logistics safety alerts.
* **Modern UI:** Clean, dark-themed interface with translucent "glass" effects.
* **Interactive Data:** Live clock and dynamic monitoring cards.
